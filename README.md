# Juego de ahorcado en JAVA

Es un juego en donde uno debe ir colocando letra a letra en la consola y tiene cierta cantidad de intentos. Si logra completar toda la palabra gana y si no, si se acaban los intentos, pierde.

## ¿Qué herramientas utilizamos?

1. Clase Scanner
2. Declaraciones y asignaciones
3. Arreglos
4. Estructura de control: Iterativa (Bucle)
5. Clase String y sus métodos valueOf y length( )
6. Clase Character y su método toLowerCase( )
7. Estructura de control: Condicional
